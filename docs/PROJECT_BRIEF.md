# SMARA Delivery MVP

## Project Overview

SMARA Delivery MVP is an internal web application developed for SMARA Catering to manage daily delivery operations.

This application is designed for internal operational use only and will be the first module of the future SMARA OS.

---

# Objective

Build a simple, fast, and reliable delivery management system that helps the operations team:

- Manage delivery orders
- Assign drivers
- Track delivery progress
- Upload Proof of Delivery (POD)
- Monitor delivery performance
- Generate operational KPI

---

# Target Users

## Owner

Can:

- View all deliveries
- Assign drivers
- Monitor delivery progress
- View KPI Dashboard

---

## Driver

Can:

- Login using PIN
- View assigned deliveries
- Open Google Maps
- Update delivery status
- Upload Proof of Delivery
